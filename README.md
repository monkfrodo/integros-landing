# Integros Landing

Landing page da newsletter — "Como construir negócios reais".

## Stack

- HTML / CSS / JavaScript (estático)
- Deploy: Vercel

## Páginas

- `index.html` — página principal
- `confirmacao.html` — confirmação de inscrição
- `destaque.html` — página de destaque
- `story.html` — página de história

## Como rodar

Abrir `index.html` no navegador ou fazer deploy estático.
